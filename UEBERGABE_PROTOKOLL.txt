7. NÄCHSTER KONKRETER SCHRITT: VERCEL DEPLOYMENT
----------------------------------------------------------------
App lokal läuft → jetzt live ins Internet bringen.

Schritt 1 — Git initialisieren & pushen (im Terminal im memberyard Ordner):
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/memberyard/memberyard.git
   git push -u origin main

Schritt 2 — Vercel Deployment:
   vercel

   Oder: vercel.com → Import → GitHub Repo auswählen

Schritt 3 — Supabase Keys in Vercel eintragen:
   Vercel Dashboard → Projekt → Settings → Environment Variables
   VITE_SUPABASE_URL     = (aus .env.local)
   VITE_SUPABASE_ANON_KEY = (aus .env.local)

Ergebnis: App ist live unter memberyard.vercel.app (oder eigener Domain)
